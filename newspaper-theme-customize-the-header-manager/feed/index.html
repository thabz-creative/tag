Thank you for your kind words, and for using our WordPress theme. If you want to create a sticky header menu, you have to open the Website Manager from the tagDiv Composer and enable the main menu sticky. When this option is set to be ON, you can preview all the settings in the tagDiv Composer list. The Transition Effect settings appear under the “Zone Specific Parameters” area. From this area you can make additional modifications, such as: none transition, slide down transition, or opacity.
